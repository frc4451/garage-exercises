# ROBOTZ Garage Programming Learning Exercises

All examples are pulled from AdvantageKit's sample projects. Credit to bases goes to:
https://github.com/Mechanical-Advantage/AdvantageKit

When cloning this repository, ensure that you have already installed WPILib and that you have a development environment for WPILib. This organization assumes you use VSCode + WPILib extensions.

Open the project by year, and open each exercise in separate VSCode windows if possible. Each exercise is its own project.
